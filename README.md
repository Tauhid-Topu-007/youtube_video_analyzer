## ✨ Key Features

| Feature | Description |
|---------|-------------|

## 🛠️ Tech Stack

### Frontend
- **Streamlit** - Interactive web application framework
- **Custom CSS** - Glass morphism design with Teal & Coral theme
- **Responsive Design** - Mobile-friendly layout

### AI & Backend
- **Agno AI** - Agent-based AI framework for structured analysis
- **Groq LLM** - High-performance language model (Qwen 3.2 32B)
- **YouTubeTools** - YouTube data extraction and processing

### Data Processing
- **Pandas** - Data manipulation for analytics dashboard
- **Python-dotenv** - Environment variable management
- **Regex** - URL parsing and validation
